It is a demo version of our main project, which is we are building. This project showcase a login form where Admin, Landlord or Customer can register and login. There Admin can manage all the accounts and monitor all the contents. Landlord can post about apartment or house for rent along with description. Customer can save, view the post also Landlord and Customer can directly message each other. Every user have a dashboard to view their activities.
this project is made with C#, winform and Microsoft SQL server. Along with implimenting the basuc crud operations and following the OOP principle it also connect to sql server to store and retrive data.
